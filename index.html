01 · Polyglot

The same greeting, everywhere

Every language has its own accent, but they all know how to say hello. Here it is in four classics — copy one, run it, and you're officially a programmer.

hello.c
/* the original */
#include <stdio.h>

int main(void) {
    printf("Hello, World!\n");
    return 0;
}

hello.rs

    println!("Hello, World!");
}
02 · Origins

A short history of two famous words

"Hello, World" wasn't planned as a tradition. It just happened — and then it never stopped happening.

1972

First sighting at Bell Labs

Brian Kernighan uses "hello, world" in an internal tutorial for the B programming language — reportedly inspired by a cartoon of a chick hatching from an egg and saying hello to the world.

1978

The C book makes it canon

The C Programming Language by Kernighan & Ritchie opens with the hello world program as its very first example. Millions of readers type it in, and a ritual is born.

1990s

Every language joins in

As Python, Java, JavaScript, and dozens of other languages emerge, each one's documentation begins the same way. Writing hello world becomes the universal first step — and an informal benchmark of how friendly a language is.

Today

Billions of hellos

It's the first commit in countless repositories, the first message from new frameworks, and the first small victory for every person learning to code. Including, quite possibly, you.
